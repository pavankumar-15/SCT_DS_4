# SkillCraft Technology - Task 4

## Exploratory Data Analysis on Marketing Campaign Dataset

This project is part of my Data Analyst Internship at SkillCraft Technology.

## Project Overview

Performed Exploratory Data Analysis (EDA) on a marketing campaign dataset to understand customer engagement, campaign performance, and return on investment (ROI) across different marketing channels.

The project focused on analyzing the reasons behind campaign success and identifying which channels should receive higher marketing budget allocation.

## Tasks Performed

* Cleaned and preprocessed the dataset
* Handled missing values and duplicate records
* Analyzed customer response patterns
* Visualized the marketing funnel
* Compared ROI across different marketing channels
* Generated insights and recommendations based on data analysis

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / Google Colab

## Analysis Features

* Customer Conversion Analysis
* ROI Comparison by Marketing Channel
* Funnel Visualization
* Campaign Performance Tracking
* Data Cleaning and Preprocessing
* Insightful Visual Reports

## Objective

The project answers the question:

“Which marketing channels should receive more budget based on ROI?”

by analyzing customer engagement, conversion rates, and campaign profitability to provide data-driven marketing recommendations.
